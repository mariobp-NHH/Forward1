What is Flask?

Flask is a micro web framework written in Python. It is classified as a microframework because it does not require particular tools or libraries.It has no database abstraction layer, form validation, or any other components where pre-existing third-party libraries provide common functions. However, Flask supports extensions that can add application features as if they were implemented in Flask itself. Extensions exist for object-relational mappers, form validation, upload handling, various open authentication technologies and several common framework related tools.

What is a Web Framework?

A Web Application Framework or a simply a Web Framework represents a collection of libraries and modules that enable web application developers to write applications without worrying about low-level details such as protocol, thread management, and so on.

  1. Create a virtual environment. A virtual environment operates as a toolbox where you can store all the tools that you need to run your app. When we add more structure and functionalities to our app, we will need more tools that we will store in our virtual environment. When we deploy our app into the server, we will give the server the list of tools stored in our virtual environment, so that it can run our application.

    A formal definition of a Python virtual environment:
    Python Virtual Environment
    To create a virtual enviroment, open a terminal in VS Code and type (python3 -m venv venv). If your python version is different type: (python -m venv venv).

  2. In VS Code, open the terminal and activate the virtual environment.
      In Windows: (venv\Scripts\activate)
      In Mac: (source venv/bin/activate)

  3. By using the terminal in VS Code, install flask: (pip install flask) (or pip3 install flask). The command pip (or pip3) depends on the Python version installed on your computer. Try both.

  4. By using the terminal in VS Code, write the code of your app, and save that file as application.py.

  5. By using the terminal in VS Code, run your App: (python application.py) (or phyton3 application.py). The command python (or python3) depends on the Python version installed on your computer. Try both.

  6. Open the app in your browser: Open the blue link that appears in VS Code after running your application.py file to open your app in the browser.

The code for our first flask app: 


from flask import Flask
application = Flask(__name__)

@application.route('/')
@application.route('/home')
def home():
    return "<h1>Home page</h1>" \
        "<p>In this project we develop 
        and app...</p>"

@application.route('/developers')
def developers():
    return "<h1>Developers</h1>" \
        "<p>In this webpage, we introduce 
        the developers of the app...</p>"

@application.route('/app_calculator')
def app_calculator():
    return "<h1>App Calculator</h1>" \
        "<p>In this webpage, we develop an 
        app to work out transport carbon e
        missions...</p>"

if __name__ == '__main__':
    application.run(debug=True)
